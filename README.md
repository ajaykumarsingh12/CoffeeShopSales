# ☕ Coffee Shop Sales Analysis | Microsoft Excel

## 📌 Project Overview

This project presents an interactive **Coffee Shop Sales Analysis Dashboard** built using **Microsoft Excel**. The dashboard transforms raw sales data into meaningful business insights through data cleaning, Pivot Tables, Pivot Charts, and interactive visualizations. It enables stakeholders to analyze sales performance, customer behavior, and store operations for informed decision-making.

## 📈 Dashboard Features

- Interactive KPI Cards
  - Total Sales
  - Total Footfall
  - Average Bill per Person
  - Average Orders per Person

- Hourly Order Analysis
  - Identifies peak business hours based on order quantity.

- Product Category Analysis
  - Sales distribution across different product categories.

- Order Size Analysis
  - Customer preferences by beverage size.

- Store Performance Analysis
  - Compares sales and footfall across store locations.

- Product Revenue Analysis
  - Highlights top revenue-generating products.

- Weekday Sales Analysis
  - Evaluates ordering trends throughout the week.

- Interactive Filters
  - Month
  - Day Name

## 🔑 Business Insights

- Peak customer orders occur during morning hours.
- Coffee products generate the highest share of total sales.
- Regular and Large beverage sizes are the most popular.
- Hell's Kitchen store records the highest overall sales.
- Barista Espresso is the top revenue-generating product.
- Weekday sales remain consistent, helping optimize staffing and inventory.
